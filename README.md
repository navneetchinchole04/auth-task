# Full Stack Auth System

A full-stack authentication system built with:

- Frontend: Next.js (App Router)
- Backend: NestJS
- Database: PostgreSQL
- Containerization: Docker & Docker Compose

## Features

- User Signup
- User Login
- Access Token & Refresh Token flow
- Secure cookies for token storage
- Table-driven unit tests (Jest)
- No external auth libraries used

## Tech Stack

| Layer | Tech |
|------|------|
| Frontend | Next.js |
| Backend | NestJS |
| Database | PostgreSQL |
| Auth | Custom implementation |
| Containers | Docker |

## Running Locally

### Prerequisites
- Docker
- Docker Compose

### Start everything

```bash```
docker compose up --build
